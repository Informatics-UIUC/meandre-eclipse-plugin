Preferences



The Meandre Component plugin can be configured by changing the preferences under Eclipse->Preferences option; Various Server parameters that can be configured are. The components are displayed from this server and are made available in the

Meande Server
  1. Meandre Server: URL including the protocol; where the Meandre Engine is running
  2. Meandre Server port: The port where the server is running.
  3. Meandre login: User login
  4. Meandre Password: Password to login

Remove Following Jars option lists the jar files that are removed from the detected dependencies. These jar files are provided by the container.
Descriptor Directory is where the descriptors are created.
Embed jars option embeds the jars in the component descriptor.
overwrite component option overwrites the component if it is already present in the server.

Package Individual Component as Jar option packages the component individually.
Other options are self explanatory.
