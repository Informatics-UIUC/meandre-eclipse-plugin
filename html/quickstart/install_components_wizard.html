Install Components Wizard

Multiple components can be installed using the Install Components Wizard. Select one or more packages that have Meandre components and select Install Components Wizard from the Meandre Components popup menu


Click on the find button to retrieve the list of components present in the pacakges. The components that have @Component Annotation will be displayed in the table. Select the components that you want to install, by clicking on the checkbox.


Click on the next button and you and in the next Wizard page click on Install to start the installation process.


Click on finish and you will get a summary of the components that were installed.
